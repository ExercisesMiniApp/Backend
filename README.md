# Backend

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit format.yml
$ yarn run format.yml

# e2e format.yml
$ yarn run format.yml:e2e

# format.yml coverage
$ yarn run format.yml:cov
```
